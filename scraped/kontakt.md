1. Om denna policy
Den här integritetspolicyn beskriver hur vi behandlar personuppgifter när du besöker vår webbplats, kontaktar oss, använder våra tjänster eller lämnar dina uppgifter via våra marknadsföringskanaler (t.ex. formulär i annonser).
2. Personuppgifter vi kan behandla
Beroende på situation kan vi behandla:
Kontaktuppgifter:
namn, e-post, telefonnummer, adress
Ärendeuppgifter:
uppgifter du lämnar i formulär, meddelanden, önskemål och annan information som behövs för att hantera din förfrågan
Kund- och avtalsuppgifter:
uppgifter som behövs för att leverera tjänst och administrera avtal
Betalnings- och fakturauppgifter:
vid köp/tjänsteleverans
Tekniska uppgifter:
IP-adress, enhet/ webbläsare, cookie-ID och användningsdata (om cookies/spårning används)
3. Varför vi behandlar personuppgifter
Vi behandlar personuppgifter för att:
besvara frågor och hantera förfrågningar
lämna offert, boka tid och följa upp dialog
leverera tjänster och administrera kundrelation
hantera fakturering, betalning och bokföring
förbättra vår webbplats, tjänster och marknadsföring (t.ex. statistik och mätning)
4. Rättslig grund
Vi stödjer behandlingen på:
Avtal
(för att leverera det du begärt eller ingå/administrera avtal)
Rättslig förpliktelse
(t.ex. bokföringskrav)
Berättigat intresse
(t.ex. kundservice, uppföljning av förfrågan, grundläggande verksamhetsutveckling)
Samtycke
(t.ex. cookies/marknadsföringsspårning där det krävs)
5. Formulär via annonser (Facebook/Instagram m.fl.)
Om du lämnar uppgifter via ett formulär i en annons (t.ex. på Facebook/Instagram) använder vi uppgifterna för att:
kontakta dig och återkoppla på din förfrågan
lämna offert/mer information och vid behov boka tid
Vi sparar uppgifterna enligt avsnitt 9 (Lagringstid).
6. Tredje parter vi kan dela personuppgifter med
Vi kan dela personuppgifter med leverantörer som hjälper oss att bedriva verksamheten, t.ex.:
IT- och driftleverantörer
(webbhotell, e-post, kommunikationsverktyg)
CRM- och kundhanteringssystem
Boknings-/affärssystem och ekonomisystem
Analys- och annonsplattformar
Meta (Facebook/Instagram)
:
Om du interagerar med våra annonser eller om vi använder Meta-verktyg för mätning/annonsering kan vissa uppgifter behandlas av Meta. Meta kan i vissa fall behandla uppgifter som självständig personuppgiftsansvarig enligt sina egna villkor. Vi delar inte personuppgifter i syfte att “sälja” dem.
(Om ni använder Meta Pixel/Conversions API – behåll denna mening, annars kan ni ta bort den):
Om du samtycker till marknadsföringscookies kan vi använda spårning/mätning för att förstå effekten av annonser och förbättra vår marknadsföring.
7. Överföring utanför EU/EES
Om våra leverantörer behandlar personuppgifter utanför EU/EES säkerställer vi lämpliga skyddsåtgärder (t.ex. standardavtalsklausuler) när det krävs.
8. Säkerhet
Vi vidtar tekniska och organisatoriska åtgärder för att skydda personuppgifter mot obehörig åtkomst, förlust och missbruk.
9. Lagringstid
Vi sparar personuppgifter endast så länge det behövs för ändamålet:
Förfrågningar/leads:
normalt i upp till [X månader] efter senaste kontakt
under avtalstiden och därefter enligt behov/krav
Bokföringsunderlag:
enligt lagkrav
Om du återkallar samtycke (där behandling bygger på samtycke) upphör vi med sådan behandling.
10. Dina rättigheter
Du har rätt att:
begära tillgång (registerutdrag)
få felaktiga uppgifter rättade
begära radering (i vissa fall)
begära begränsning eller invända mot behandling
begära dataportabilitet (i vissa fall)
återkalla samtycke när behandling bygger på samtycke
Kontakta oss på
[E-post]
för att utöva dina rättigheter. Du kan även klaga till Integritetsskyddsmyndigheten (IMY).
11. Ändringar i policyn
Vi kan uppdatera policyn. Den senaste versionen finns alltid på denna sida med uppdateringsdatum.
